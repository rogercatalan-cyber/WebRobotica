Programacio

Explicacio de l'algoritme, lectura de sensors, control de motors i fragments de codi mes importants.

Introduccio

El programa combina una lectura continua del sensor de distancia amb decisions simples: avançar quan hi ha espai, detenirse davant un obstacle i girar per buscar una ruta lliure.

Contingut del document

Pega aqui el HTML exportat des de Google Docs amb explicacions, pseudocodi o captures de l'IDE.

Exemple de logica

  1. Mesurar distancia amb el sensor frontal.
  2. Si la distancia es segura, activar ambdos motors cap endavant.
  3. Si la distancia es baixa, parar, retrocedir breument i girar.
  4. Repetir el cicle fins completar la prova.
