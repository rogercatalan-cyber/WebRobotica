# Documentos exportados desde Google Docs

Guarda aqui los HTML descargados desde Google Docs:

Archivo -> Descargar -> Pagina web (.html, comprimido)

Despues puedes copiar el contenido del archivo HTML exportado y pegarlo en la zona marcada con:

```html
<!-- INICIO GOOGLE DOCS -->
<!-- FIN GOOGLE DOCS -->
```

Si el ZIP incluye imagenes, copialas a `img/` o a una subcarpeta dentro de `docs/` y revisa las rutas `src` para que funcionen al abrir la web localmente.
